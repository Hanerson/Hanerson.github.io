# 个人生活博客 README

## 博客简介
这是一份以白描方式记录日常的个人生活博客，无刻意修饰，仅如实留存生活里的细碎瞬间与真实感触。内容围绕日常起居、学习实践、饮食体验、细碎感悟等展开，既记录编程学习、赛事参与等成长片段，也捕捉美食、运动、休闲娱乐等生活日常，偶尔夹杂对周遭事物的简单思考，是对平凡日子的朴素存档。

## 内容范畴
- 日常碎片：涵盖学习（编程、算法赛事等）、生活（饮食、运动、休闲、出行）等日常点滴记录；
- 零散感悟：偶尔记录对生活、网络、成长等方面的简单思考与细碎想法；
- 轻量分类：内容按自然风景、美食探店、旅行记忆、名言警句、艺术创作、日常碎片等维度简单归类，标签则围绕治愈、励志、创意、难忘等真实感受标注。

## 记录特点
无固定更新节奏，随生活轨迹零散记录；内容以真实、简洁为核心，不刻意渲染情绪，仅以直白的方式留存当下的状态与感受。

## 总结
1. 博客核心是**如实记录日常**，涵盖学习、生活、感悟等多维度的平凡瞬间；
2. 语言风格以**白描、简洁**为主，无过度修辞，不刻意突出某一具体事件；
3. 内容有基础分类与标签体系，是对个人生活状态的朴素存档。
